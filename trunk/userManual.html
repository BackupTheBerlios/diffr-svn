Запуск программы
Для запуска программы необходимо из каталога ./bin выполнить команду java –cp “.” de.belios.diffr.Init ../
В Windows это делает файл ./start.bat
Окно программы
Окно программы делится на две части. Справа находится изображение поверхности и падающего поля. Левая часть состоит из трёх вкладок: Input data, Result и Algorithm. Вкладка Input data используется для изменения входных данных. В нутри неё ещё две вкладки – Surface (Для изменения поверхности) и Impinging field (Для изменения падающего поля). Вкадка Result используется для просмотра результата вычислений. Состоит из Reflected field (Отражённое поле), Passed field (Прошедшее поле) и Surface current (Токи в поверхности). Если выбранный алгоритм не умеет считать какую-нибудь из частей результата, вкладка для неё показываться не будет. Вкладка Algorithm используется для выбора алгоритма и изменения его параметров.
Задача (Task)
Задача – это объединение задаваемых входных данных (поверхность и падающее поле), результата (если он посчитан) и текущего алгоритма (вместе параметрами).
Тип задачи (Task type)
Каждая задача относится к одному из типов задач. В типе задачи хранится информация о том, какие классы надо использовать для хранения поверхности и падающего поля, а так же какие алгоритмы можно использовать для данной задачи.
Состояние
В нижней части окна показывается текущее состояние.
Оно может быть одним из:
Result isn`t calculated
Для текущих входных данных результат не посчитан.
Result is calculated
Результат посчитан.
Task is solving…
Результат считается в данный момент.
Task is stopped
Пользователь остановил вичисление.
Error
Во время работы алгоритма произошла ошибка. Возможно алгоритм не предусмотрен для работаты с заданными входными данными.
Меню Fail
New task
Создать новую задачу, с входными данными по умолчанию. Текущая задача при этом будет потеряна.
Load task
Загрузить ранее сохранённую задачу. Текущая задача при этом будет потеряна.
Save task
Сохранить текущую задачу.
Exit
Выйти в Windows. Текущая задача будет сохранена в файл “autosave.task” и автоматически загружена при следующем запуске программы.
Меню Task
Start
Начать вычисление результата.
Stop
Прервать вычисление результата.
Меню Algorithms
Add algorithm for current task
Добавить новый алгоритм для данного типа задач.
Remove algorithm from current task
Убрать ранее добавленный алгоритм.

Масштаб изображения
В левом верхнем углу изображения поверхности и падающего поля есть надпись “Scale:” + число. Это масштаб изображения – сколько пикселов приходится на единицу измерения (сантиметр).
Тип задачи “Diffraction of plane wave on periodic surface”
Это единственный на данный момент тип задач.
Поверхность – периодическая поверхность, её форма задаётся коэффициентами Фурье.
Падающее поле – плоская волна, задаётся углом к вертикали, длиной волны, амплитудой и поляризацией.
