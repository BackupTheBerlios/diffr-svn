Что это такое

Двумерные задачи дифракции световых, электромагнитных и звуковых волн описываются хорошо известным уравнением Гельмгольца с различными дополнительными условиями на границах раздела сред и условиями на поведение поля в бесконечности.

Особенностью этих задач является то, что кроме нескольких вырожденных случаев у этих задач нет аналитических решений. В настоящее время разработано большое количество численных и асимптотических методов решения этих задач. Эти методы являются разнородными, зависят от различных дополнительных параметров и предположений, «работают» только в определенном диапазоне параметров физической задачи, могут вычислять не все характеристики решения.

В мире имеется большое количество исследователей и научных школ, развивающих свои методы решения двумерных задач дифракции. Однако, в силу разнородности существующих алгоритмов и программ, для решения конкретной задачи оказывается крайне сложно найти подходящий метод, по тем же причинам очень сложно сравнить различные существующие алгоритмы.

Diffr6 - единая среда для моделирования двумерных задач дифракции. В этой среде исследователи могут добавлять свои алгоритмы и моделировать при помощи них задачи дифракции.

Данная среда позволит для конкретных областей значений параметров задачи дифракции определить наиболее эффективные приближенные методы. Также среда позволит сравнить различные методы для различных классов параметров физической задачи.

Запуск программы
Для запуска программы необходимо из каталога ./bin выполнить команду java –cp “.” de.belios.diffr.Init ../
В Windows это делает файл ./start.bat
Задача (Task)
Задача – это объединение задаваемых входных данных (поверхность и падающее поле), результата (если он посчитан) и текущего алгоритма (вместе параметрами).
Окно программы
Окно программы делится на две части. Справа находится изображение поверхности и падающего поля. Левая часть состоит из трёх вкладок: Input data, Result и Algorithm. Вкладка Input data используется для изменения входных данных. В нутри неё ещё две вкладки – Surface (Для изменения поверхности) и Incident field (Для изменения падающего поля). Вкадка Result используется для просмотра результата вычислений. Состоит из Reflected field (Отражённое поле), Passed field (Прошедшее поле) и Surface current (Токи в поверхности). Если выбранный алгоритм не умеет считать какую-нибудь из частей результата, вкладка для неё показываться не будет. Вкладка Algorithm используется для выбора алгоритма и изменения его параметров.
Поверхность (Surface)

Поверхность всегда периодическая. Задаётся формой поверхности (вкладка Shape) и проводимостью (вкладка Conductivity).

Форма поверхности задаётся периодом, параметром “shift” и коэффициентами Фурье. Высота поверхности в точке x вычисляется по формуле:
Сумма по всем n меньше k (An * cos(n * x * 2pi / p) + Bn * sin(n * x * 2pi / p)) + s ,
где pi - число пи; p – период; k – количество коеффициентов Фурье; An – n-ый коеффициент при косинусе; Bn – n-ый коеффициент при синусе; s – параметр “shift”.
Падающее поле (Incident wave)

Падающее поле – это плоская электромагнитная волна. Задаётся углом к вертикали (в градусах), длиной волны(в сантиметрах), амплитудой и поляризацией (E или H).
Отражённое поле (Reflected field)

Это одна из состовляющих результата. Состоит из нескольких плоских волн. Возможно 2 варианта просмотра – текстовый и графический.
Токи в поверхности (Surface current)

Это одна из состовляющих результата. Комплексная функция от координаты на поверхности. Изображается в виде двух кривых – модуль и фаза.
Состояние
В нижней части окна показывается текущее состояние.
Оно может быть одним из:
Result isn`t calculated
Для текущих входных данных результат не посчитан.
Result is calculated
Результат посчитан.
Task is solving…
Результат считается в данный момент.
Task is stopped
Пользователь остановил вичисление.
Input data not supported
Алгоритм не предусмотрен для работаты с заданными входными данными.
Меню Fail
New task
Создать новую задачу, с входными данными по умолчанию. Текущая задача при этом будет потеряна.
Load task
Загрузить ранее сохранённую задачу. Текущая задача при этом будет потеряна.
Save task
Сохранить текущую задачу.
Exit
Выйти в Windows. Текущая задача будет сохранена в файл “autosave.task” и автоматически загружена при следующем запуске программы.
Меню Task
Start
Начать вычисление результата.
Stop
Прервать вычисление результата.
Меню Algorithms
Add algorithm
Добавить новый алгоритм.
Remove algorithm
Убрать ранее добавленный алгоритм.
Меню Help
User manual
Руководство пользователя.
Масштаб изображения
В левом верхнем углу изображения поверхности и падающего поля есть надпись “Scale:” + число. Это масштаб изображения – сколько пикселов приходится на единицу измерения (сантиметр).
